# 🤖 AI Requirements & Backlog Intelligence System

A Streamlit web app powered by Groq AI (LLaMA 3) that transforms
requirements documents into actionable product backlog artifacts.

## Features
- 📝 User Story Generator
- 🔍 Requirements Gap Analyzer  
- 💬 Conversational Backlog Assistant

## Tech Stack
Python · Streamlit · Groq API (LLaMA 3) · ChromaDB · Sentence Transformers

## Run Locally
1. pip install -r requirements.txt
2. Create .env file → add GROQ_API_KEY=your_key
3. streamlit run chat_assistant.py
